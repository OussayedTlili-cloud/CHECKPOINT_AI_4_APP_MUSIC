# Specification du projet - Mini Lecteur Deezer

## Objectif
Creer une mini-application web permettant de rechercher un artiste, d’afficher ses titres via l’API Deezer et d’ecouter un extrait audio.

## Fonctionnalites
1. Page d’accueil :
   - Champ de recherche
   - Bouton Rechercher

2. Page des resultats :
   - Liste des musiques (titre, artiste, image)
   - Lecteur audio pour chaque musique
   - Mise en page en grille
   - Bouton Retour vers la recherche

## Technologies requises
- HTML
- CSS
- JavaScript
- API Deezer via RapidAPI

## Contraintes
- Aucun framework externe
- Application 100 % front-end
- Code organise : separation HTML/CSS/JS
- Interface responsive
