## Presentation
Ce projet est une mini-application web permettant de rechercher un artiste, d’afficher ses titres et d’écouter un extrait audio de 30 secondes en utilisant l’API Deezer via RapidAPI.

## Fonctionnalites
- Recherche d’artistes par mot-clé
- Affichage des titres dans une grille
- Lecteur audio intégré
- Bouton pour revenir a la page de recherche
- Interface responsive
- HTML, CSS et JavaScript uniquement

## Technologies utilisees
- HTML5  
- CSS3  
- JavaScript Vanilla  
- API Deezer (RapidAPI)
